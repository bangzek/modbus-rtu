modbus-rtu
==========

[![Go Report Card](https://goreportcard.com/badge/github.com/bangzek/modbus-rtu)](https://goreportcard.com/report/github.com/bangzek/modbus-rtu)
[![Go Reference](https://pkg.go.dev/badge/github.com/bangzek/modbus-rtu.svg)](https://pkg.go.dev/github.com/bangzek/modbus-rtu)


modbus-rtu (rtu) is library for using ModBus RTU protocol in Go. Currently it
only support Modbus Master for controlling ModBus Slave/Device. So you can make
software that can talk to device that using ModBus via serial port.
